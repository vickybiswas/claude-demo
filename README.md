## Running all from one command line
I talked about High-level outcome, Constraining details, and Validation methods.
Today let's build a frontend (simple HTML/CSS/JS) and backend (python docker) using agents.
This time it's one central tool (claude) doing everything and you will be not have to visit any other tool.
As promised earlier no costs involved to try things out.

We will directlly push High-level outcome as our instruction.
We will put Constraining details as our config (CLAUDE.md files).
We will create Validation methods as our skills (SKILLS.md files).

This is not to establish best practice but as an eye opener for nay sayers.
We will do it for:
* [A new project](#new-project)
* [An existing project](#extablished-projects)


## Walkthrough
