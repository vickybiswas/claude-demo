# CAPS Folder - Python Best Practices

## Variable Naming Convention
- ALL VARIABLES AND CONSTANTS MUST BE IN UPPERCASE
- Use underscores to separate words
- Example: `MAX_RETRIES = 5`, `API_KEY = "abc123"`, `USER_ID = 123`

## Constants
- Use uppercase for all constants
- Example: `PI = 3.14`, `GRAVITY = 9.8`

## Variables
- Even regular variables should be uppercase
- Example: `CURRENT_USER = "admin"`, `TOTAL_COUNT = 0`
